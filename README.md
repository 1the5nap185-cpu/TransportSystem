
## Особенности реализации

### 1. Запечатанные классы (Sealed Classes)
- Класс Vehicle запечатан и разрешает наследование только LandTransport, AirTransport, WaterTransport
- Каждый из этих классов также запечатан и разрешает конкретные реализации

### 2. Перечисления (Enum)
- FuelType - типы топлива (бензин, дизель, электричество и т.д.)

### 3. Абстрактные классы и интерфейсы
- Интерфейс Transport определяет общий контракт
- Абстрактный класс Vehicle реализует базовую функциональность

### 4. Конкретные классы транспорта
-  Car - автомобиль
- Motorcycle - мотоцикл
- Airplane - самолет
- Ship - корабль
